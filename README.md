# vimium

### Custom key mappings
```shell
unmapAll

map <a-s> Vomnibar.activateTabSelection
map j scrollDown
map k scrollUp
map h scrollLeft
map l scrollRight

map gg scrollToTop
map G scrollToBottom

map i enterInsertMode
map v enterVisualMode
map gi focusInput

map / enterFindMode
map n performFind
map N performBackwardsFind
```
